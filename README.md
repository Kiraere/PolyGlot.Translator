# Додаток-словник (Англійсько-Український)

Готовий до використання веб-додаток для англійсько-українського та українсько-англійського перекладу з підтримкою синонімів, автодоповнення та кешування.

## 🚀 Функції
- **Двонаправлений переклад**: EN → UA та UA → EN.
- **Автодоповнення**: Підказки під час введення (мінімум 2 символи).
- **Синоніми**: Відображення споріднених слів для кращого контексту.
- **Кешування**: Високопродуктивний пошук та переклад за допомогою Caffeine cache.
- **Постійне зберігання**: Використовує базу даних H2 в пам'яті (легко переключити на PostgreSQL/MySQL).
- **Чиста архітектура**: Дотримується принципів SOLID та найкращих практик Spring Boot.

## 🛠️ Стек технологій
- **Backend**: Java 17, Spring Boot 3, Spring Data JPA, H2, Caffeine Cache.
- **Frontend**: Next.js 14, React, Lucide React (іконки).

## 📋 Попередні вимоги
- Java 17 або вище
- Maven 3.8+
- Node.js 18+ (для фронтенду)

## 🏃 Початок роботи

### 1. Запустіть Backend
```bash
cd backend
mvn spring-boot:run
```
Сервер запуститься на `http://localhost:8080`.

### 2. Запустіть Frontend
```bash
cd frontend
npm install
npm run dev
```
Додаток буде доступний за адресою `http://localhost:3000`.

### 3. Завантажте тестові дані
Після того, як обидві частини запущені, відкрийте браузер, перейдіть до додатка та натисніть **"Load Sample Data"** внизу сторінки, щоб заповнити словник.

## 🔍 Приклади API

### Завантажити словник
**POST** `/api/load`
```json
[
  {
    "word": "hello",
    "translations": ["привіт", "вітаю"],
    "synonyms": ["hi", "greetings"]
  }
]
```

### Перекласти
**GET** `/api/translate?word=hello&direction=en-ua`

### Автодоповнення
**GET** `/api/search?query=he`

## 🏗️ Структура проєкту
- `backend/`: Додаток Spring Boot.
- `frontend/`: Додаток Next.js.
- `sample-data.json`: Початкові дані для тестування.
